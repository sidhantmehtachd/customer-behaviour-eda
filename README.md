# customer-behaviour-eda
End-to-end customer behavior analysis project using Python, SQL, and Power BI. Includes data cleaning, feature engineering, customer segmentation, and an interactive dashboard to uncover sales trends and business insights.


## 📌 Overview

This project focuses on performing Exploratory Data Analysis (EDA) on customer shopping data using Python, SQL, and Power BI.
The goal is to understand customer behavior, identify patterns, and generate actionable business insights through analysis and visualization.

---

## 📂 Dataset

The dataset contains customer transaction and behavioral information such as:

* Customer demographics (age, gender)
* Purchase details (items, amount, frequency)
* Subscription status
* Discounts and shipping preferences

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy)
* **MySQL** (Data querying and analysis)
* **Power BI** (Dashboard and visualization)

---

## 🔄 Project Workflow

### 1. Data Loading (Python)

* Imported dataset into Python environment
* Performed initial inspection and validation

### 2. Data Cleaning & Preprocessing

* Handled missing values
* Converted data types
* Created new features (e.g., age groups, segments)

### 3. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends
* Identified key patterns in customer behavior
* Generated summary statistics

### 4. SQL Analysis (MySQL)

Executed analytical queries to extract business insights, including:

* Revenue comparison by gender
* High-value customers using discounts
* Top-rated products
* Shipping type vs spending
* Subscription vs non-subscription behavior
* Discount usage rate per product
* Customer segmentation (New, Returning, Loyal)
* Top products within each category using window functions
* Revenue contribution by age group

* Loyal customers contribute significantly to total revenue
* Discount usage influences purchasing behavior
* Certain product categories dominate sales
* Subscription status impacts average spending
